# 福利彩票预测系统配置指南

## 环境变量配置

### DeepSeek API Key

预测系统使用DeepSeek大模型进行AI分析，需要配置API密钥：

1. 访问 [DeepSeek官网](https://www.deepseek.com/) 注册账号
2. 获取API Key
3. 在环境变量中配置：

```bash
# .env.local (开发环境)
DEEPSEEK_API_KEY=your_deepseek_api_key_here
```

```bash
# Vercel环境变量（生产环境）
DEEPSEEK_API_KEY=your_deepseek_api_key_here
```

**注意**：如果没有配置DeepSeek API Key，系统会自动使用基于统计分析的备用预测方案，功能仍然可用，但不会有AI分析结果。

## 安装依赖

```bash
npm install
```

新增的依赖：
- `@radix-ui/react-tooltip` - Tooltip组件（用于显示业务逻辑和技术说明）

## 功能说明

### 1. 统计分析
- **频率分析**：统计每个号码的出现频率，识别热号、冷号、温号
- **遗漏值分析**：计算每个号码距离上次出现的期数
- **分布分析**：分析号码的区间、奇偶、大小、和值、跨度分布
- **模式识别**：识别连号、周期性模式、组合模式

### 2. AI分析（DeepSeek）
- 使用DeepSeek大模型分析历史数据中的复杂模式
- 生成AI预测建议和推理过程
- 提供预测理由和置信度

### 3. 机器学习
- 特征工程：提取频率、遗漏、分布等特征
- 概率模型：计算每个号码的出现概率
- 多策略预测：生成保守型、平衡型、激进型预测

### 4. 综合预测
- 融合三种方法的预测结果
- 生成5组最优预测号码
- 提供详细的预测理由和置信度

## 使用方式

1. 访问 `/lottery/predict` 页面
2. 设置历史数据期数（建议100期）
3. 点击"生成预测"按钮
4. 查看5组预测号码和分析结果

## 重要声明

⚠️ **彩票本质是随机事件，本系统仅用于数据分析和学习目的，不能保证中奖。**

- 系统通过数据分析提高预测的合理性
- 可以适当提高小奖概率，但大奖概率极低
- 请理性购彩，量力而行
