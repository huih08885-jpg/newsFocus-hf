# 关键词功能快速开始指南

## 🚀 3步完成关键词匹配

### 第1步：配置关键词组 ✅（已完成）
- 你已经创建了"电商"关键词组
- 确认关键词组已**启用**（显示绿色"启用"标签）

### 第2步：获取新闻数据并匹配

**选项 A：自动匹配（推荐）**
1. 访问首页或 `/platforms` 页面
2. 点击 **"立即爬取"** 按钮
3. 等待爬取完成
4. 系统会自动匹配最近1小时的新闻

**选项 B：手动匹配所有新闻**
1. 访问 `/settings/keywords` 页面
2. 点击右上角 **"手动匹配所有新闻"** 按钮
3. 等待匹配完成
4. 查看匹配结果提示

### 第3步：查看匹配结果
1. 访问 **"新闻列表"** 页面（左侧导航栏）
2. 匹配的新闻会显示：
   - ✅ 关键词组标签（如 "电商"）
   - ✅ 权重分数
   - ✅ 匹配次数
3. 使用筛选功能：
   - 在筛选栏选择 **"电商"**
   - 只显示匹配该词组的新闻

## 📊 验证匹配是否成功

### 方法1：查看新闻列表
- 访问 `/news` 页面
- 查看新闻是否显示"电商"标签
- 如果有标签，说明匹配成功 ✅

### 方法2：使用筛选功能
- 在新闻列表页面，选择"电商"关键词组
- 如果能看到新闻，说明匹配成功 ✅

### 方法3：运行诊断工具
```bash
pnpm test:keyword
```
这会显示详细的匹配统计信息

## 💡 提示

1. **先测试再使用**：使用"测试"功能验证关键词组是否正确
2. **定期运行匹配**：每次爬取后，匹配会自动执行
3. **修改关键词后**：记得点击"手动匹配所有新闻"重新匹配

## ❓ 常见问题

**Q: 为什么看不到匹配结果？**
- 确认关键词组已启用
- 确认有新闻数据（运行爬取）
- 确认执行了匹配（手动或自动）

**Q: 匹配需要多长时间？**
- 通常每1000条新闻需要几秒钟
- 匹配过程不会阻塞界面

**Q: 如何匹配历史新闻？**
- 点击"手动匹配所有新闻"按钮
- 系统会匹配数据库中所有新闻
